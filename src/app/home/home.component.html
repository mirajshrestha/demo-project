Beginner Steps
Installing Angular-CLI

First you need to have node.js and npm installed in your machine. Then you can use npm to install the cli by using following command in command line:
npm install -g @angular/cli

Beginner Steps
Generating a new project with Angular-Cli

We can generate a new project use ng new command. This will create your project folder with default files in your directory.
ng new project-name

Beginner Steps
Creating Component

Components are the Building Blocks of Angular. It will create you .css, .html, . spec.ts, .ts files in your component folder. You can create a component using following command:
ng g component component-name

Beginner Steps
Running your Angular Application with the CLI

The Cli helps you to start your application in your default browser using the following command.
ng serve
